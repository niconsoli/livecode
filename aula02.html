Aula 02: Estruturas Condicionais (if, else e switch)

Programas não apenas processam dados, eles tomam decisões! As estruturas condicionais em Java permitem que o seu código execute diferentes blocos de instrução dependendo se uma condição é verdadeira (`true`) ou falsa (`false`).

Operadores de Comparação

Para construir a condição, usamos operadores de comparação. A condição sempre deve retornar um valor booleano (`true` ou `false`):

Operador Significado Exemplo Resultado (se x=10)
== Igual a x == 10 true
!= Diferente de x != 5 true
> Maior que x > 10 false
< Menor que x < 15 true
>= Maior ou igual a x >= 10 true
<= Menor ou igual a x <= 9 false

A Estrutura if e else

A instrução if executa um bloco de código se a condição for verdadeira. O else é opcional e executa um bloco alternativo se a condição for falsa.

int nota = 7; if (nota >= 7) { System.out.println("Parabéns! Você foi aprovado."); } else { System.out.println("Infelizmente, você precisa estudar mais."); }

Múltiplas Condições (else if)

Para testar múltiplas possibilidades em sequência, usamos o else if:

int hora = 14; if (hora < 12) { System.out.println("Bom dia!"); } else if (hora < 18) { System.out.println("Boa tarde!"); } else { System.out.println("Boa noite!"); }

Operadores Lógicos

Podemos combinar múltiplas condições usando operadores lógicos:

  • && (AND lógico): Verdadeiro se TODAS as condições forem verdadeiras.
  • || (OR lógico): Verdadeiro se PELO MENOS UMA das condições for verdadeira.
  • ! (NOT lógico): Inverte o valor booleano.
int idade = 17; boolean possuiHabilitacao = false; if (idade >= 18 && possuiHabilitacao) { System.out.println("Pode dirigir."); } else { System.out.println("Não pode dirigir."); }

Com as estruturas condicionais, seus programas podem tomar decisões complexas e responder de forma inteligente a diferentes situações. Na próxima aula, aprenderemos a automatizar tarefas repetitivas com laços de repetição.
