Aula 04: Entrada de Dados com a Classe Scanner

Até agora, nossos programas utilizavam dados fixos. Para criar aplicações interativas, precisamos que o usuário possa fornecer informações. Em Java, a classe Scanner é a ferramenta mais comum para ler entradas do teclado.

Passo 1: Importar a Classe

A classe `Scanner` não é carregada automaticamente. Você precisa importá-la para usá-la. Isso deve ser a primeira linha do seu arquivo (depois do `package`, se houver):

import java.util.Scanner;

Passo 2: Criar o Objeto Scanner

Dentro do método `main`, você cria uma instância do `Scanner`, indicando que ele deve ler a entrada do sistema (`System.in`):

Scanner scanner = new Scanner(System.in);

Passo 3: Capturar Diferentes Tipos de Entrada

O objeto `Scanner` possui diferentes métodos para ler diferentes tipos de dados:

  • nextLine(): Lê a linha inteira de texto, incluindo espaços.
  • nextInt(): Lê um número inteiro (`int`).
  • nextDouble(): Lê um número decimal (`double`).
  • next(): Lê apenas a próxima palavra (até o próximo espaço).

Exemplo Completo de Interação

Veja um exemplo que combina a captura de texto e número:

import java.util.Scanner; public class InteracaoUsuario { public static void main(String[] args) { Scanner scanner = new Scanner(System.in); System.out.print("Qual é o seu nome? "); String nome = scanner.nextLine(); System.out.print("Qual é a sua idade? "); int idade = scanner.nextInt(); System.out.println("Olá, " + nome + "! Você tem " + idade + " anos."); scanner.close(); } }

Atenção: Fechando o Scanner

É uma boa prática fechar o objeto `Scanner` quando você terminar de usá-lo, chamando o método `scanner.close()` para liberar recursos do sistema.
