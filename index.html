Aulas de Fundamentos Java

Aula 01: Variáveis e Tipos

Introdução aos tipos de dados primitivos (int, double, boolean) e como declarar, inicializar e usar variáveis para armazenar dados em Java. É o alicerce para qualquer programa.

Aula 02: Condicionais (if/else)

Aprenda a tomar decisões complexas. Você verá como usar operadores lógicos e a estrutura `if`, `else if` e `else` para controlar o fluxo de execução do seu programa com base em condições booleanas.

Aula 03: Laços de Repetição (for/while)

Conceitos de repetição com os loops `for` e `while`. Você aprenderá a criar estruturas que permitem executar um bloco de código várias vezes, facilitando tarefas repetitivas e otimizando o desenvolvimento de programas.
