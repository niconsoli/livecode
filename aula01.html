Aula 01: Variáveis e Tipos de Dados em Java

Esta é a base de qualquer programa: como armazenamos e manipulamos informações. Em Java, as variáveis são como caixas rotuladas que guardam dados, e cada "caixa" precisa ter um "tipo" definido para saber qual tipo de informação ela pode conter (números, texto, etc.).

O que são Variáveis?

Uma variável é uma área de memória reservada para armazenar um valor. Para usar uma variável, precisamos primeiro declarar ela. A declaração define o tipo de dado que a variável irá guardar e o nome que usaremos para acessá-la.

Tipos de Dados Primitivos

Java possui oito tipos primitivos, mas vamos focar nos mais comuns para iniciarmos:

  • int: Usado para números inteiros (sem casas decimais), como $-10$, $0$, $42$.
  • double: Usado para números de ponto flutuante (com casas decimais), como $3.14$, $10.0$.
  • char: Usado para armazenar um único caractere, como 'A', '7', '$'.
  • boolean: Usado para armazenar valores de verdade, ou seja, apenas true ou false.

Declarando e Inicializando Variáveis

A sintaxe básica para declarar uma variável é: `tipo nomeDaVariavel;`. Para dar um valor inicial, usamos o operador de atribuição (`=`):

int idade = 25; double preco = 19.99; char inicial = 'J'; boolean estaLogado = true; int numeroDeAlunos; numeroDeAlunos = 15;

Exemplo Prático

Veja como podemos usar variáveis em um pequeno programa para calcular a área de um círculo (usando $\pi \approx 3.14$):

public class CalculadoraArea { public static void main(String[] args) { int raio = 5; double PI = 3.14; double area = PI * raio * raio; System.out.println("O raio é: " + raio); System.out.println("A área do círculo é: " + area); } }

Entender variáveis e tipos é o primeiro passo para criar programas que armazenam dados do mundo real. Na próxima aula, veremos como tomar decisões com esses dados usando estruturas condicionais!
