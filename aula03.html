Aula 03: Laços de Repetição (Loops for e while)

A repetição de tarefas é uma das maiores vantagens da programação. Em Java, usamos os laços de repetição (loops) para executar o mesmo bloco de código várias vezes, economizando linhas e otimizando o seu programa.

O Laço for (Para contagens definidas)

O `for` é ideal quando você sabe exatamente quantas vezes a repetição deve ocorrer. Ele exige três partes em sua definição, separadas por ponto e vírgula:

  1. Inicialização: Executada apenas uma vez no início (ex: `int i = 0`).
  2. Condição: Avaliada a cada repetição. Se for verdadeira, o loop continua. Se for falsa, o loop para.
  3. Incremento/Decremento: Executado após cada repetição (ex: `i++` ou `i--`).
for (int i = 1; i <= 5; i++) { System.out.println("Contagem: " + i); }

O Laço while (Enquanto uma condição for verdadeira)

O `while` é usado quando você não sabe de antemão quantas vezes a repetição ocorrerá. Ele executa o bloco de código enquanto a condição especificada for verdadeira.

É crucial que a condição se torne falsa em algum momento dentro do loop, caso contrário, você terá um Loop Infinito, que travará o programa.

int contador = 0; while (contador < 3) { System.out.println("Repetição número: " + (contador + 1)); contador++; }

Exemplo Prático: Tabuada com for

Vamos usar o `for` para gerar a tabuada de multiplicação de um número:

int numero = 8; System.out.println("Tabuada de " + numero + ":"); for (int i = 1; i <= 10; i++) { int resultado = numero * i; System.out.println(numero + " x " + i + " = " + resultado); }

Com os laços `for` e `while`, você pode automatizar tarefas como processar listas de dados, simular eventos ou repetir ações até que o usuário atinja um objetivo. Na próxima aula, combinaremos tudo isso para permitir a entrada de dados pelo usuário, tornando seus programas interativos!
